# Document Similarity Detector

A Python project that detects similarity between text documents using:

- Shingling
- MinHash
- Locality Sensitive Hashing (LSH)
- Jaccard Similarity

## Features

- Generates document shingles
- Computes MinHash signatures
- Uses LSH for similarity detection
- Calculates exact Jaccard similarity score

## Technologies

- Python
- hashlib
